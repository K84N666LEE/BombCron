무료 설치


DB 기반 다중 잡 오케스트레이션
 SQL 한 줄 – INSERT INTO jobs … 만으로 새 작업이 즉시 등록 · 가동됩니다.

3‑Way 트리거
  ① URL : REST / GraphQL / Webhook 모두 호출 가능
  ② Java 클래스·메서드 : com.company.batch.InvoiceJob::run() 식 지정
  ③ 스크립트 : Shell · Python · JS 파일 경로 지정 후 자동 실행

CRON & human‑readable 일정
  0 0/5 * * * * 뿐 아니라 every 5 minutes처럼도 등록 → 내부에서 CRON으로 변환

대시보드 + Slack / Teams 알림
  웹 UI에서 실행 이력·다음 실행 시각을 실시간 확인, 실패 시 알림 전송

외부 호출(Webhook‑in)
  타 시스템이 Bomb Scheduler URL을 호출 → 즉시 잡 실행 또는 토큰 기반 승인 후 실행

경량 설치
  Spring Boot 2.x 단일 .jar, H2/PG/MySQL 모두 지원. Docker 이미지 제공.
